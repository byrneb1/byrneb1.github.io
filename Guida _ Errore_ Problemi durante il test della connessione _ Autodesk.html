Errore: Problemi durante il test della connessione

Durante il test della connessione se si verifica uno di questi scenari di errore, seguire il collegamento corrispondente per risolvere il problema:

  • Error AADSTS50105 - Sorry, but we're having trouble signing you in
  • Sign in error - SAMLRequest or SAMLResponse must be present as query string parameters in HTTP request for SAML Redirect binding
  • This page can’t be found
  • Error message with an error Code: GENERAL_NONSUCCESS

Se gli scenari precedenti non consentono di risolvere l'errore, provare ad eseguire una traccia SAML. Per istruzioni dettagliate, vedere Come eseguire una traccia SAML
